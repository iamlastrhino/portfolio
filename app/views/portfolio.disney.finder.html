The Project

Walt Disney Parks and Resorts Online has developed a shared platform in an effort to support and serve all the theme park websites throughout the world. This project was for the "Finder" portion of the web platform. In a quick summary, Finder is made up of the search, listing, description, maps and detail pages for all attractions, parks, resorts, hotels and events.

While I was the technical lead on this project my goals and accomplishments were to support internationalization, improve responsive/adaptive compatibility, add accesibilty compliance, improve performance and technical integration with the existing platform.
