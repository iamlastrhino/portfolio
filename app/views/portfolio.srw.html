The Project

Seattle Restaurant Week is the collaboration of many great local restaurants, sponsored by local businesses and hosted/facilitated by The Seattle Times. Before this website was developed the process was completely manual. A list of restaurants with links to their pages on SeattleTimes.com was compiled and published.

The goal for this project was to change this to be a much easier, efficient and streamlined production proccess, while improving the user experience. The producer now simply generates a JSON data file that is published and all of the handy listing, filtering and links are handled on the client side using JQuery.
